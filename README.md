# Bright Library Sing-Up Page

## Description
This project is a Front-End Sign-Up page for a hypothetical library named Bright Library. It is inspired by The Odin Project's "Intermediate HTML and CSS Sign-Up Form" Assignment and is built to practice form creation and user-input validation via HTML. 

## Features
- HTML validation
- Modern Sign-Up layout 
- Beautiful colors

## Technologies Used
- HTML
- CSS

## Project Structure
```
Bright-library-signup-form/
├── index.html
├── style.css
└── image.jpg
```
## How to Run
1. Clone or download the repository
2. Open `index.html` in your browser
3. Or click the live preview

From this simple yet important project, I learned how easy and useful it is to validate user input using just HTML. Additionally, I learned the importance of picking the right color for a web page and how it affects the overall design.
